# Trainers Search App  
A full-stack web application built using **Django REST Framework** (Backend) and **React** (Frontend).  
The app allows users to **search, add, update, and list trainers** with secure authentication.

---

## Features

### Backend (Django REST Framework)
- JWT Authentication (SimpleJWT)
- Trainer CRUD API (Add, Search, Update, Delete)
- Pagination & filtering support
- CORS configuration for React
- MySQL / SQLite database support

### Frontend (React)
- Login authentication
- Search trainers
- Add trainer form
- Trainer list with update/delete
- React Router navigation
- Axios for API requests
- Responsive UI
